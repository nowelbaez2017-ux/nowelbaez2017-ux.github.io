# Login Moderno de Prueba Nowel

Interfaz moderna de login y registro creada con HTML, CSS y JavaScript.

## Vista general

- Pantalla de inicio de sesion.
- Pantalla de registro.
- Animacion lateral para cambiar entre ambos formularios.
- Iconos sociales con Font Awesome.

## Tecnologias

- HTML5
- CSS3
- JavaScript
- Font Awesome

## Como verlo

Abre `index.html` en el navegador.
